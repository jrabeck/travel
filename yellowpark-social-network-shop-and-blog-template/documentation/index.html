Template Installation

  1. FTP Upload:
    • Open up your FTP manager and connect to your hosting
    • Browse to required directory (Normally public_html)
    • Upload the files inside yellowpark folder.

Pages

    • index.html is for Default Home page
    • shop.html is for shop page
    • shop-category.html is for shop category page
    • shop-category-fullwidth.html is for shop category fullwidth page
    • single-product.html is for product details page
    • map.html is for full width map page
    • map-sidebar.html is for map with sidebar page
    • map-sidebar-2.html is for map with sidebar 2 page
    • map-sidebar-3.html is for map with sidebar 3 page
    • map-sidebar-4.html is for map with sidebar 4 page
    • map-sidebar-5.html is for map with sidebar 5 page
    • map-sidebar-6.html is for map with sidebar 6 page
    • blog.html is for blog page
    • blog-details.html is for blog details page
